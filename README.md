# EigIdleSlots - Casino Slot Idle Game

Ein Browser-basiertes Slot-Machine Idle Game mit tiefgreifender Progression, Bonuses und Mini-Games.

## 🎮 Features

### Slot-Maschinen (3 Stück)
- **Classic Slots** - Vegas-style Symbole (Cherry, Lemon, Bell, 7s, Bar)
- **Fruit Slots** - Frucht-Themen (freischaltbar)
- **Lucky 7s** - Glücksspiel-Themen (freischaltbar)

### Progression System
- **Auto-Spinner** (+1 Spin pro Sekunde) - ab 100 Coins
- **Coin Multiplier** (+25% Coins pro Level) - ab 500 Coins
- **Spin Speed** (schnellere Animation) - ab 1.000 Coins
- **Neue Maschine** (freischaltbare Slot-Maschinen) - ab 5.000 Coins

### Bonus Features
- **Free Spins** - Automatisch bei Big Wins, manuell nutzbar
- **Magic Jackpot** - Progress-basiertes Jackpot (100 Progress = ~5.000 Coins)
- **Glücksrad** - 1.000 Coins für Chance auf bis zu 100.000 Coins
- **Prestige Reset** - 10.000 Coins Minimum für permanente x2 Bonus-Effekte

### Mini-Games
- **Lottery** (500 Coins) - Zahlen-Raten, Gewinn: 10.000 Coins
- **Münzwurf** (500 Coins) - 50/50 Chance, Gewinn: 1.000 Coins
- **Risk Game** (1.000 Coins) - Verdoppeln oder verlieren
- **Lucky Wheel** (2.000 Coins) - Großes Gewinn-Potenzial

### Weitere Features
- **Offline Progress** - Spins laufen weiter wenn du nicht online bist
- **Coin Rain** - Zufällige Boni alle 45 Sekunden
- **Floating Coins Animation** - Visuelle Effekte bei Big Wins
- **Auto-Save** alle 30 Sekunden (LocalStorage)
- **Täglicher Bonus** - Jeden Tag neue Coins
