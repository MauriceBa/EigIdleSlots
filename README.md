# EigIdleSlots - Casino Slot Idle Game

Ein Browser-basiertes Slot-Machine Idle Game mit Progression, Bonuses und Offline-Fortschritt.

## Features

- **3 Slot-Maschinen**: Classic, Fruit & Lucky 7s
- **Progression System**: Upgrades für Auto-Spinner, Coin Multiplier, Spin Speed
- **Bonus Features**: Free Spins, Magic Jackpot, Glücksrad
- **Prestige System**: Reset für permanente Bonuseffekte
- **Save/Load**: Automatisches Speichern im Browser (LocalStorage)
- **Offline Progress**: Fortschritt während AFK

## Installation

```bash
git clone https://github.com/MauriceBa/EigIdleSlots.git
cd EigIdleSlots
# Einfach mit jedem Webserver oder direkt im Browser öffnen
```
